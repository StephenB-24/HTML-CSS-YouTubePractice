# HTML CSS YouTube Practice
A remake of a YouTube page for practicing HTML and CSS

This project was done for practice with:
- Basic HTML
- Basic CSS
- Nested Layouts
- CSS Grid
- Flexboxes
- Position Types (for tooltips)
